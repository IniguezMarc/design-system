# Getting Started

Welcome to **Marc's Design System** (`@iniguezmarc/design-system`). A modern, accessible, and type-safe React component library.

## ÔøΩ Installation

Install the package using NPM:

```bash
npm install @iniguezmarc/design-system
```

## ‚öôÔ∏è Setup

### 1. Import Styles
You **must** import the library's CSS in your application's entry point (e.g., `main.tsx`, `App.tsx`) to ensure components are styled correctly.

```tsx
import '@iniguezmarc/design-system/dist/style.css';
```

### 2. Usage
Import components directly from the package:

```tsx
import { Button, ProjectCard } from '@iniguezmarc/design-system';

function App() {
  return (
    <div className="p-10">
      <Button label="Click Me" variant="primary" />

      <ProjectCard
         title="My Project"
         description="A cool project using the design system."
         tags={['React', 'TypeScript']}
         image="https://via.placeholder.com/300"
      />
    </div>
  );
}
```

## üß© Library Structure

The library follows **Atomic Design** principles:

- **Atoms**: Basic building blocks (Buttons, Inputs, Badges, Avatars).
- **Molecules**: Simple groups of atoms (Project Cards).
- **Organisms**: Complex UI sections (Navbar, Hero, Footer, ProfileSection).

## üõ† Developer Guide

If you want to run this project locally for development:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/IniguezMarc/design-system.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start Storybook:**
   ```bash
   npm run storybook
   ```
   Visit `http://localhost:6006` to interact with components.
