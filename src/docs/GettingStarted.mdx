This will start the Storybook server (usually at `http://localhost:6006`), where you can explore:

- **Molecules**: Combinations of atoms (Cards, Search bars).
- **Organisms**: Complex sections (Navbar, Hero, Footer).

## ðŸ“¦ Usage in External Projects

If you want to use a component from this library in another project:

1.  **Copy the Code**: The easiest way is to copy the component file (e.g., `Button.tsx`) and its dependencies (like `index.css` or Tailwind config) directly into your project.
2.  **Dependencies**: Ensure your project has the required dependencies installed (React, Tailwind CSS, etc.).
3.  **Import**: Import the component in your project:
    ```tsx
    import { Button } from './components/atoms/Button/Button';
    ```

> **Note**: In the future, this library could be published as an NPM package for easier installation (`npm install @marc-design-system/ui`).
