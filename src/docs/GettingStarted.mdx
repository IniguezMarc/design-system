# Getting Started

Welcome to **Marc's Design System** (`@iniguezmarc/design-system`). A modern, accessible, and type-safe React component library.

Install the package using NPM:

```bash
npm install @iniguezmarc/design-system
```

### 1. Import Styles
You **must** import the library's CSS in your application's entry point (e.g., `main.tsx`, `App.tsx`) to ensure components are styled correctly.

```tsx
import '@iniguezmarc/design-system/dist/style.css';
```

### 2. Usage
Import components directly from the package:

```tsx
import { Navbar, Hero, ContentGrid, Footer } from '@iniguezmarc/design-system';

function App() {
  const projects = [
    {
      id: '1',
      title: 'E-commerce App',
      description: 'A modern shopping platform built with React.',
      image: 'https://via.placeholder.com/600x400',
      tags: ['React', 'TypeScript'],
      url: 'https://github.com/my-user/project-1'
    }
  ];

  return (
    <div className="min-h-screen bg-gray-50 dark:bg-gray-900">
      <Navbar 
        logo="My Portfolio" 
        links={[{ label: 'Home', href: '#' }, { label: 'Projects', href: '#projects' }]} 
      />

      <main>
        <Hero
          title="Building Digital Experiences"
          subtitle="Full Stack Developer & UI Designer"
          // Customize colors easily via props
          greetingColor="text-indigo-600"
          actions={[
            { label: "View Work", variant: "primary", onClick: () => console.log('Scroll') }
          ]}
        />

        <div id="projects">
          <ContentGrid
            title="Latest Projects"
            items={projects}
            layout="grid"
            itemButtonLabel="View Case Study"
            // Dark mode overrides
            darkTitleColor="dark:text-indigo-100"
          />
        </div>
      </main>

      <Footer copyrightOwner="Marc I√±iguez" />
    </div>
  );
}
```

## üß© Library Structure

The library follows **Atomic Design** principles:

- **Atoms**: Basic building blocks (Buttons, Inputs, Badges, Avatars).
- **Molecules**: Simple groups of atoms (Project Cards).
- **Organisms**: Complex UI sections (Navbar, Hero, Footer, ProfileSection).

## üõ† Developer Guide

If you want to run this project locally for development:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/IniguezMarc/design-system.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start Storybook:**
   ```bash
   npm run storybook
   ```
